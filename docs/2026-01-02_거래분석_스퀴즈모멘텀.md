## 📊 스퀴즈 모멘텀 전략 개요

### 1. 핵심 개념 (존 카터의 Squeeze Momentum Pro)

**스퀴즈(Squeeze)**란 변동성이 극도로 수축된 상태를 의미합니다:
- **볼린저 밴드(BB)**: 가격 변동성 측정 (20일 평균 ± 2σ)
- **켈트너 채널(KC)**: ATR 기반 변동성 측정 (20일 EMA ± 1.5 ATR)
- **Squeeze On**: BB가 KC 안으로 들어감 → 변동성 극도 수축 → 곧 폭발 예상
- **Squeeze Off**: BB가 KC 밖으로 나감 → 변동성 확대 시작

### 2. 매수 조건 (4가지 필수)

```
✅ 조건 1: Squeeze On (BB < KC)
   → 볼린저 상단 < 켈트너 상단
   → 볼린저 하단 > 켈트너 하단

✅ 조건 2: Momentum Up (2-3봉 연속 상승)
   → 모멘텀 히스토그램이 연속 증가

✅ 조건 3: 현재가 > VWAP
   → 거래량 가중 평균가 위에서 거래

✅ 조건 4: 히스토그램 색상 = Bright Green 🟢
   → 모멘텀 > 0 AND 모멘텀 증가 중
```

### 3. 매도 조건 (색상 기반 단계적 청산)

#### 🟢 Bright Green (밝은 녹색)
- **의미**: 모멘텀 > 0 이고 **가속 중**
- **액션**: **절대 보유** - 매도 금지!
- **이유**: 상승 추세 가속화, 최대 수익 구간

#### 🟡 Dark Green (어두운 녹색)
- **의미**: 모멘텀 > 0 이지만 **감속 중**
- **액션**: **부분 익절** (30% 매도)
- **조건**: 수익률 1% 이상일 때
- **이유**: 모멘텀 둔화, 조정 가능성

#### 🔴 Dark Red (어두운 빨강)
- **의미**: 모멘텀 < 0 이고 **하락 가속**
- **액션**: **전량 청산**
- **조건**: 수익률 0.5% 이상일 때
- **이유**: 추세 반전 확정

#### 🟠 Bright Red (밝은 빨강)
- **의미**: 모멘텀 < 0 이지만 **하락 둔화**
- **액션**: **전량 청산** (반등 기대 안함)
- **조건**: 수익률 0.5% 이상일 때
